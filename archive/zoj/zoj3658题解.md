# ZOJ 3658 Simple Function 题解

## 1. 问题形式

给定五个整数 `A, B, C, D, E`，求函数的值域：

```text
f(x) = (Ax^2 + Bx + C) / (x^2 + Dx + E)
```

记：

```text
P(x) = Ax^2 + Bx + C
Q(x) = x^2 + Dx + E
```

原函数的定义域要求：

```text
Q(x) != 0
```

---

## 2. 核心思路：反解 x

直接对函数求导可以找到极值，但这题还会出现竖直渐近线、空洞点、退化方程等情况，单靠求导很容易漏。

更稳的做法是反过来考虑：

> 一个实数 `y` 是否属于值域？

令：

```text
y = P(x) / Q(x)
```

交叉相乘：

```text
yQ(x) = P(x)
```

整理：

```text
P(x) - yQ(x) = 0
```

即：

```text
(A - y)x^2 + (B - Dy)x + (C - Ey) = 0
```

因此：

```text
y 在值域中
<=> 存在实数 x，使得 P(x) - yQ(x) = 0，并且 Q(x) != 0
```

这句话是整题的核心。

---

## 3. 候选值域

先忽略 `Q(x) != 0` 的限制，只要求下面这个关于 `x` 的方程有实数解：

```text
(A - y)x^2 + (B - Dy)x + (C - Ey) = 0
```

当它是二次方程时，需要判别式非负：

```text
(B - Dy)^2 - 4(A - y)(C - Ey) >= 0
```

展开得到：

```text
(D^2 - 4E)y^2 + (4AE + 4C - 2BD)y + (B^2 - 4AC) >= 0
```

记：

```text
alpha = D^2 - 4E
beta  = 4AE + 4C - 2BD
gamma = B^2 - 4AC
```

于是候选值域来自：

```text
alpha * y^2 + beta * y + gamma >= 0
```

根据 `alpha, beta, gamma` 解这个一次或二次不等式，得到候选区间集合。

注意：这里得到的只是“候选值域”，不是最终值域。后面还要检查原函数的定义域限制。

---

## 4. 常数函数特判

如果：

```text
B = AD
C = AE
```

那么：

```text
P(x) = A * Q(x)
```

因此在所有合法 `x` 上：

```text
f(x) 恒等于 A
```

值域直接是：

```text
[A, A]
```

这个特判应该放在最前面，避免后续判别式退化成 `0 >= 0` 时误判为全体实数。

例如：

```text
A = B = C = D = E = 0
```

函数是：

```text
f(x) = 0 / x^2
```

定义域是：

```text
x != 0
```

值域是：

```text
[0, 0]
```

而不是全体实数。

---

## 5. 候选区间之后要删除不可达点

判别式法只得到“候选值域”。

有些 `y` 虽然让反解方程有实数解，但这些解可能全部满足：

```text
Q(x) = 0
```

这种 `y` 实际上不能取到，需要从候选区间中删除。

因此需要一个统一判断函数：

```text
hasLegalSolution(y)
```

它判断：

```text
P(x) - yQ(x) = 0
```

是否存在至少一个实数解 `x`，并且满足：

```text
Q(x) != 0
```

如果不存在合法解，就把这个 `y` 从候选区间里挖掉。

---

## 6. 只需要检查两类可疑点

不需要枚举所有 `y`。只需要检查两类可疑点。

### 6.1 检查 y = A

因为当：

```text
y = A
```

时，反解方程：

```text
(A - y)x^2 + (B - Dy)x + (C - Ey) = 0
```

会退化，一些情况下 `y = A` 只是水平渐近线，未必能取到。

所以直接调用：

```text
digIfNotReachable(A)
```

含义是：如果 `A` 这个值没有合法 `x` 可以取到，就从候选区间中删除 `A`。

### 6.2 检查分子分母公共根造成的空洞值

解分母：

```text
Q(r) = 0
```

如果：

```text
P(r) != 0
```

那么 `r` 只是竖直渐近线，不会造成某个有限值缺失。

如果：

```text
P(r) = 0
```

那么 `r` 是分子分母公共根，可能造成空洞值。

如果：

```text
Q'(r) != 0
```

也就是分母在 `r` 处是单根，那么空洞候选值是：

```text
yr = P'(r) / Q'(r)
   = (2Ar + B) / (2r + D)
```

然后调用：

```text
digIfNotReachable(yr)
```

如果：

```text
Q'(r) = 0
```

说明分母是重根。此时不能使用 `P'(r) / Q'(r)`。

继续看：

```text
P'(r) = 2Ar + B
```

如果：

```text
P'(r) = 0
```

说明分子也是二重根，整体约掉后得到常数 `A`，所以检查：

```text
digIfNotReachable(A)
```

否则不需要删除有限值。

## 7. hasLegalSolution(y) 的判断逻辑

令：

```text
R(x) = P(x) - yQ(x)
     = (A - y)x^2 + (B - Dy)x + (C - Ey)
```

目标是判断：

```text
R(x) = 0
```

是否有合法实根。所谓合法，就是这个根还要满足：

```text
Q(x) != 0
```

### 7.1 二次方程

如果：

```text
A - y != 0
```

则 `R(x) = 0` 是二次方程。

计算判别式：

```text
delta = (B - Dy)^2 - 4(A - y)(C - Ey)
```

如果：

```text
delta < 0
```

无实根，返回 `false`。

如果：

```text
delta = 0
```

检查唯一根是否满足：

```text
Q(x) != 0
```

如果满足，返回 `true`，否则返回 `false`。

如果：

```text
delta > 0
```

检查两个根。只要有一个满足：

```text
Q(x) != 0
```

就返回 `true`。只有两个根都非法时才返回 `false`。

### 7.2 一次方程

如果：

```text
A - y = 0
B - Dy != 0
```

则 `R(x) = 0` 是一次方程。

唯一解是：

```text
x = -(C - Ey) / (B - Dy)
```

如果：

```text
Q(x) != 0
```

返回 `true`，否则返回 `false`。

### 7.3 常数方程

如果：

```text
A - y = 0
B - Dy = 0
```

则只剩：

```text
C - Ey = 0
```

如果：

```text
C - Ey != 0
```

无解，返回 `false`。

如果：

```text
C - Ey = 0
```

说明：

```text
P(x) - yQ(x) 恒等于 0
```

因为 `Q(x)` 是二次多项式，最多只有两个非法点，所以总能找到合法 `x`，返回 `true`。

## 8. 区间挖洞

如果某个 `y` 不可达，就从当前候选区间中删除它。

例如候选区间是：

```text
(-INF, INF)
```

挖掉：

```text
1
3/2
```

得到：

```text
(-INF, 1) U (1, 1.5) U (1.5, INF)
```

挖洞时注意：

```text
1. 洞在区间内部：拆成两个开区间。
2. 洞在闭端点：把该端点改成开端点。
3. 挖洞后过滤空区间。
4. 浮点比较使用 eps。
5. 输出时避免 -0.0000。
```

---

## 9. 典型例子

### 例 1

```text
A = 0, B = 1, C = 0, D = 1, E = -10
```

函数：

```text
f(x) = x / (x^2 + x - 10)
```

候选不等式为：

```text
41y^2 - 2y + 1 >= 0
```

这个二次式的判别式小于 0，且开口向上，所以恒成立。

值域为：

```text
(-INF, INF)
```

### 例 2

```text
A = 1, B = 3, C = 0, D = 2, E = 0
```

函数：

```text
f(x) = (x^2 + 3x) / (x^2 + 2x)
     = x(x + 3) / x(x + 2)
```

定义域：

```text
x != 0
x != -2
```

形式上约分后：

```text
f(x) = (x + 3) / (x + 2)
     = 1 + 1 / (x + 2)
```

所以：

```text
y = 1
```

取不到。

同时 `x = 0` 是空洞点，对应缺失值：

```text
(0 + 3) / (0 + 2) = 3/2
```

最终值域：

```text
R \ {1, 3/2}
```

也就是：

```text
(-INF, 1) U (1, 3/2) U (3/2, INF)
```

## 10. 易错点总结

```text
1. 不能只求导，竖直渐近线会影响值域形状。
2. 判别式不等式只是候选值域，不是最终值域。
3. 分母为 0 的 x 不能作为合法解。
4. 分子分母公共根会造成空洞值。
5. y = A 必须单独检查。
6. 常数函数必须最前面特判。
7. 反解方程可能退化成一次或常数方程，不能无脑套二次公式。
8. 挖洞后要过滤空区间。
9. Java 中注意整数除法，例如 -gamma / beta。
10. 输出时注意 -0.0000。
```

最后再强调一遍，本题的关键不是直接研究 `f(x)` 的图像，而是反过来判断每个 `y` 是否存在合法原像 `x`：

```text
y 在值域中
<=> P(x) - yQ(x) = 0 至少有一个实根 x，并且 Q(x) != 0
```
