# HDU 4425 Fantastic Necklace 解法总结

> 一句话：**唯一环函数图找环 + 环外入树同构 DP + 环上 Burnside/Polya 计数**。

这份笔记是基于最终通过版代码整理的复盘，重点记录算法模型、`dp` 语义、公式来源、实现细节和几个调试时踩过的坑。

---

## 1. 题目模型

输入可以看成一个函数图：每个点都有且只有一条出边。

```java
graph[i] = p;
addEdge(p, i);   // 反图：p -> i
inDegree[p]++;
```

题目保证整张图只有一个环，所以结构是：

```text
若干入树挂在一个有向环上

        tree        tree
          \        /
           cycle - cycle
          /        \
       tree        tree
```

因此可以分两部分处理：

1. 环外部分是若干 rooted tree，用树同构 DP 处理。
2. 环上部分有旋转等价，用 Burnside/Polya 处理。

---

## 2. 找唯一环

函数图找环的做法是拓扑删叶子。

先把所有入度为 `0` 的点入队：

```java
for (int i = 1; i <= n; i++) {
    if (inDegree[i] == 0) queue.offer(i);
}
```

然后不断删除非环点：

```java
while (!queue.isEmpty()) {
    int top = queue.poll();

    int p = graph[top];
    inDegree[p]--;
    if (inDegree[p] == 0) queue.offer(p);
}
```

最后：

```text
inDegree[i] > 0 的点就是环上的点
inDegree[i] == 0 的点就是环外入树节点
```

找环时从任意一个 `inDegree[i] > 0` 的点出发，沿着 `graph` 走一圈即可：

```java
List<Integer> cycle = new ArrayList<>();
int pt = 0;
for (int i = 1; i <= n; i++) {
    if (inDegree[i] > 0) {
        pt = i;
        break;
    }
}

for (int i = pt; ; ) {
    cycle.add(i);
    i = graph[i];
    if (i == pt) break;
}
```

## 3. 树同构 DP

### 3.1 `treehash[u]` 的含义

`treehash[u]` 用来表示以 `u` 为根的 rooted tree 的同构类型。

对于一个点 `u`，它的所有儿子子树是无序的，所以要先收集儿子的 `treehash`，排序，然后合并：

```java
int count = 0;
for (int e = head[top]; e != -1; e = nextEdge[e]) {
    int u = to[e];
    tempHashes[count++] = treehash[u];
}

Arrays.sort(tempHashes, 0, count);

for (int i = 0; i < count; i++) {
    treehash[top] += shift(tempHashes[i]);
}
```

其中：

```java
long shift(long x) {
    x ^= x << 13;
    x ^= x >>> 7;
    x ^= x << 17;
    return x;
}
```

这不是严格无碰撞的树同构编号，但这题数据可以接受。

更严格的做法是：

```text
排序后的儿子类型数组 -> Map 离散化成唯一 typeId
```

不过最终通过代码采用的是 `long treehash`。

---

### 3.2 `dp[u]` 的含义

这题最容易错的是 `dp[u]` 的语义。最终代码里分成两类。

#### 非环节点

非环节点的 `dp[u]` 表示：

```text
以 u 为根的入树，在 u 的父亲颜色已经固定时，
u 必须与父亲颜色不同的方案数。
```

因此根节点 `u` 自身有：

```text
m - 1
```

种颜色选择。

所以非环节点转移为：

```java
dp[top] = (dpNew * m_minus_1) % mod;
```

---

#### 环上节点

环上节点的颜色不能单独相对父亲决定，因为环上的颜色要整体计数。

所以环上节点的 `dp[u]` 表示：

```text
只考虑挂在 u 下面的环外树枝的方案数，不包含 u 自身颜色。
```

因此环上节点转移为：

```java
dp[top] = dpNew;
```

这里**不能乘 `m - 1`**。

---

## 4. 为什么同构儿子要用组合数

假设某个点下面有 `c` 个完全同构的儿子子树，每棵这种子树有 `a` 种本质不同的染色方案。

如果这些儿子有顺序，方案数是：

```text
a^c
```

但在 rooted tree 同构意义下，这些同构儿子互相不可区分，所以要数的是：

```text
从 a 种方案里可重复选 c 个的多重集合数量
```

即：

```text
C(a + c - 1, c)
```

代码里对应：

```java
long getComp(long a, long b) {
    long up = 1;
    long down = 1;
    for (int i = 1; i <= b; i++) {
        up = (up * ((a + b - i) % mod)) % mod;
        down = (down * i) % mod;
    }
    return (up * power(down, mod - 2, mod)) % mod;
}
```

因此树 DP 的核心是：

```text
dpNew = Π C(dp[childType] + count[childType] - 1, count[childType])
```

代码中通过排序后的 `tempHashes` 分组：

```java
long dpNew = 1;
for (int i = 0; i < count; ) {
    int j = i;
    while (j < count && tempHashes[j] == tempHashes[i]) j++;

    int c = j - i;
    long w = globalDps.get(tempHashes[i]);

    dpNew = (dpNew * getComp(w, c)) % mod;
    i = j;
}
```

## 5. 环上处理

处理环上点时，只统计环外挂树：

```java
for (int e = head[top]; e != -1; e = nextEdge[e]) {
    int u = to[e];
    if (inDegree[u] == 0) {
        tempHashes[count++] = treehash[u];
    }
}
```

其中：

```text
inDegree[u] == 0 代表 u 是环外点
inDegree[u] != 0 代表 u 是环上点
```

环上点的 `treehash[top]` 要包含它挂着的树枝，用于判断旋转后结构是否一致。

环上点的 `dp[top]` 只表示挂树贡献，不包含环点自身颜色：

```java
dp[top] = dpNew;
```

特别注意：环上点的 `dp` 语义和非环点不同，所以不要把环上点写回 `globalDps` 覆盖非环树型的方案数。

---

## 6. Burnside / Polya 处理环旋转

设环长为：

```java
int cyclesize = cycle.size();
```

necklace 有旋转等价，所以要对旋转群取平均。

枚举 `cyclesize` 的约数 `d`：

```java
for (int d : getDivisors(cyclesize)) {
    ...
}
```

这里 `d` 可以理解为某类旋转下的轨道数量，也可以理解为被保留下来的周期长度。

---

### 6.1 判断这个旋转是否有效

若旋转后一个方案要保持不变，那么同一轨道上的环点挂树结构必须一致。

代码中判断：

```java
boolean valid = true;
for (int i = 0; i < d; i++) {
    long expectedHash = treehash[cycle.get(i)];
    long expectedDP = dp[cycle.get(i)];

    for (int j = i + d; j < cyclesize; j += d) {
        if (treehash[cycle.get(j)] != expectedHash || dp[cycle.get(j)] != expectedDP) {
            valid = false;
            break;
        }
    }

    if (!valid) break;
}

if (!valid) continue;
```

含义：

```text
cycle[i], cycle[i+d], cycle[i+2d], ... 必须拥有相同的挂树类型和挂树方案数。
```

如果不一致，这类旋转没有固定方案，贡献为 `0`。

---

### 6.2 长度为 `d` 的环染色数

长度为 `d` 的环，相邻颜色不同，使用 `m` 种颜色，染色方案数为：

```text
(m - 1)^d + (-1)^d * (m - 1)
```

代码里：

```java
long term2 = (d % 2 == 1) ? (mod - m_minus_1) : m_minus_1;
long cycleColorWays = (power(m_minus_1, d, mod) + term2) % mod;
```

也就是：

```text
d 为奇数： (m - 1)^d - (m - 1)
d 为偶数： (m - 1)^d + (m - 1)
```

---

### 6.3 乘上挂树贡献

当前周期长度为 `d`，只需要乘前 `d` 个代表点的挂树贡献：

```java
long totalWaysForK = cycleColorWays;
for (int i = 0; i < d; i++) {
    totalWaysForK = (totalWaysForK * dp[cycle.get(i)]) % mod;
}
```

---

### 6.4 为什么乘 `phi(cyclesize / d)`

旋转 `k` 步后，环被划分出的轨道数量是：

```text
gcd(cyclesize, k)
```

当：

```text
gcd(cyclesize, k) = d
```

这样的 `k` 的数量是：

```text
phi(cyclesize / d)
```

所以：

```java
long countK = getPhi(cyclesize / d);
long contribution = (totalWaysForK * (countK % mod)) % mod;
ans = (ans + contribution) % mod;
validcyclesize += countK;
```

最后 Burnside 平均：

```java
ans = (ans * power(validcyclesize, mod - 2, mod)) % mod;
```

这里 `validcyclesize` 是所有有效旋转数量。

## 7. 实现细节和踩坑记录

### 7.1 MLE 的主要来源

早期版本用了：

```java
List<Integer>[] backgraph
List<Long> subTreehash
Queue<Integer> queue = new LinkedList<>()
HashMap<Long, ...>
new Thread(..., 1 << 27)
```

其中 `ArrayList[]`、`Long/Integer` 装箱、`LinkedList` 节点、过大的线程栈都会增加内存压力。

最终代码把反图改成链式前向星：

```java
int[] head = new int[maxn];
int[] to = new int[maxn];
int[] nextEdge = new int[maxn];
```

加边：

```java
void addEdge(int u, int v) {
    to[edgeCount] = v;
    nextEdge[edgeCount] = head[u];
    head[u] = edgeCount;
    edgeCount++;
}
```

`tempHashes` 也改成原生数组：

```java
long[] tempHashes = new long[maxn];
```

这样内存稳定在 `O(n)`。

---

### 7.2 `globalDps` 覆盖问题

非环点的 `dp` 乘了 `m - 1`，环上点的 `dp` 没有乘 `m - 1`。

所以同一个 `treehash`，环上点和非环点的 `dp` 语义不同。

错误写法：

```java
dp[top] = dpNew;
globalDps.put(treehash[top], dp[top]);  // 环上点不应该写回
```

正确做法：

```java
dp[top] = dpNew;
```

即环上点不要写回 `globalDps`。

---

### 7.3 `treehash` 理论上可能碰撞

最终 AC 代码用 `long treehash` 判断 rooted tree 同构。比赛数据可以过，但理论上存在碰撞风险。

严格写法应该是：

```text
每个节点的类型 = sorted(所有儿子的类型编号)
把这个数组作为 key 离散化为 typeId
```

这样就不依赖 hash 随机性。

---

### 7.4 Java 提交类名

HDU 通常要求主类名为：

```java
public class Main
```

本地文件里是：

```java
public class FantasticNecklace
```

提交时如果 CE，就把类名改成 `Main`。

## 8. 复杂度

找环：

```text
O(n)
```

树 DP 部分，每个节点会排序自己的儿子：

```text
Σ sort(deg[u])
```

上界可以理解为：

```text
O(n log n)
```

环上 Burnside 枚举约数，每个约数最多扫一遍环：

```text
O(τ(cyclesize) * cyclesize)
```

其中 `τ` 是约数个数，通常很小。

总复杂度：

```text
O(n log n)
```

空间复杂度：

```text
O(n)
```

## 9. 最终记忆版

```text
HDU 4425 = 唯一环函数图 + 树同构 DP + Burnside。

1. 拓扑删叶子，剩下唯一环。
2. 环外从叶子向环做 rooted tree DP。
3. 同构儿子按无序多重集合计数：C(dp+c-1,c)。
4. 非环点 dp 乘 m-1，环上点 dp 不乘。
5. 环上枚举旋转周期 d，检查同轨道挂树是否一致。
6. 长度 d 的合法环染色数：
   (m-1)^d + (-1)^d(m-1)。
7. 每类旋转数量是 phi(cyclesize/d)。
8. Burnside 平均得到答案。
```

## 10. 最终通过代码

```java
import java.io.*;
import java.util.*;

public class FantasticNecklace {
    public static void main(String[] args) {
        new Thread(null, new Runnable() {
            public void run() {
                new Solution().run();
            }
        }, "run", 1 << 25).start();
    }
}

class Solution {
    
    Kattio io;

    void run() {
        io = new Kattio();
        solve();
        io.flush();
    }

    long mod = 1000000007;
    private static final int maxn = 100005;
    int edgeCount = 0;
    int[] head = new int[maxn];
    int[] to = new int[maxn];
    int[] nextEdge = new int[maxn];
    long[] tempHashes = new long[maxn];
    Map<Long, Long> globalDps = new HashMap<>();

    void addEdge(int u, int v) {
        to[edgeCount] = v;
        nextEdge[edgeCount] = head[u];
        head[u] = edgeCount;
        edgeCount++;
    }

    /* 核心逻辑 */
    void solve() {
        while (io.hasNext()) {
            // 读入
            long n = io.nextLong(), m = io.nextLong()%mod;
            long m_minus_1 = (m-1+mod)%mod;
            int[] graph = new int[(int)n+1];
            int[] inDegree = new int[(int)n+1];
            Arrays.fill(head,0,(int)n+1,-1);
            edgeCount = 0;
            for (int i = 1; i <= n; i++) {
                int p = io.nextInt();
                // 正图，只有一个出边
                graph[i] = p;
                // 反图
                addEdge(p, i);
                inDegree[p]+=1;
            }
            // 拓扑排序找环
            Queue<Integer> queue = new LinkedList<>();
            // 入度为0入队
            for (int i = 1; i <= n; i++) if (inDegree[i] == 0) queue.offer(i);
            long[] treehash = new long[(int)n+1];
            Arrays.fill(treehash, 1L);
            globalDps.clear();
            // globalDps.put(1L,1L);
            long[] dp = new long[(int)n+1];
            while (!queue.isEmpty()) {
                int top = queue.poll();
                // 融合top的treehash
                int count = 0;
                for (int e = head[top]; e != -1; e = nextEdge[e]) {
                    int u = to[e];
                    tempHashes[count++] = treehash[u];
                }
                Arrays.sort(tempHashes, 0, count);
                for (int i = 0; i < count; i++) {
                    treehash[top] += shift(tempHashes[i]);
                }
                // 找一样的treehash，统计个数，并计算新dp
                long dpNew = 1;
                for (int i = 0; i < count; ) {
                    int j = i;
                    while (j < count && tempHashes[j] == tempHashes[i]) j++;
                    int c = j-i;
                    // 子树的hash值
                    long w = globalDps.get(tempHashes[i]);
                    dpNew = (dpNew * getComp(w, c)) % mod;
                    i = j;
                }
                dp[top] = (dpNew*m_minus_1)%mod;
                globalDps.put(treehash[top], dp[top]);
                // 减入度
                int p = graph[top];
                inDegree[p]--;
                if (inDegree[p] == 0) queue.offer(p);
            }
            // 找主环
            List<Integer> cycle = new ArrayList<>();
            int pt = 0;
            for (int i = 1; i <= n; i++) if (inDegree[i] > 0) {
                pt = i; break;
            }
            for (int i = pt;;) {
                cycle.add(i);
                i = graph[i];
                if (i == pt) break;
            }
            int cyclesize = cycle.size();
            // 遍历环，找同样的方案
            for (int k = 0; k < cyclesize; k++) {
                int top = cycle.get(k);
                // 融合top的treehash
                int count = 0;
                for (int e = head[top]; e != -1; e = nextEdge[e]) {
                    int u = to[e];
                    if (inDegree[u] == 0) tempHashes[count++] = treehash[u];
                }
                Arrays.sort(tempHashes, 0, count);
                for (int i = 0; i < count; i++) {
                    treehash[top] += shift(tempHashes[i]);
                }
                // 找一样的treehash，统计个数，并计算新dp
                long dpNew = 1;
                for (int i = 0; i < count; ) {
                    int j = i;
                    while (j < count && tempHashes[j] == tempHashes[i]) j++;
                    int c = j-i;
                    // 子树的hash值
                    long w = globalDps.get(tempHashes[i]);
                    dpNew = (dpNew * getComp(w, c)) % mod;
                    i = j;
                }
                dp[top] = dpNew;
            }
            // Polya计数定理
            long ans = 0;
            long validcyclesize = 0;
            for (int d : getDivisors(cyclesize)) {
                // 旋转k步之后，环会被切分成k个独立的轨道
                // 轨道内的树hash必须一致
                boolean valid = true;
                for (int i = 0; i < d; i++) {
                    long expectedHash = treehash[cycle.get(i)];
                    long expectedDP = dp[cycle.get(i)];
                    for (int j = i+d; j < cyclesize; j+=d) {
                        if (treehash[cycle.get(j)] != expectedHash || dp[cycle.get(j)] != expectedDP) {
                            valid = false;
                            break;
                        }
                    }
                    if (!valid) break;
                }
                if (!valid) continue;
                // 给d的环染色
                long term2 = (d%2==1)?(mod-m_minus_1):m_minus_1;
                long cycleColorWays = (power(m_minus_1,d,mod)+term2) % mod;
                // 累加树枝的贡献
                long totalWaysForK = cycleColorWays;
                for (int i = 0; i < d; i++) totalWaysForK = (totalWaysForK * dp[cycle.get(i)]) % mod;
                long countK = getPhi(cyclesize/d);
                validcyclesize += countK;
                long contribution = (totalWaysForK*(countK%mod))%mod;
                ans = (ans + contribution) % mod;
            }
            ans = (ans * power(validcyclesize, mod-2, mod)) % mod;
            io.println(ans);
        }
    }

    int gcd(int a, int b) {
        return b == 0 ? a : gcd(b, a%b);
    }

    long getPhi(int n) {
        long res = n;
        for (int i = 2; i * i <= n; i++) {
            if (n % i == 0) {
                res = res / i * (i - 1);
                while (n % i == 0) n /= i;
            }
        }
        if (n > 1) res = res / n * (n - 1);
        return res;
    }

    List<Integer> getDivisors(int n) {
        List<Integer> divisors = new ArrayList<>();
        for (int i = 1; i * i <= n; i++) {
            if (n % i == 0) {
                divisors.add(i);
                if (i * i != n) {
                    divisors.add(n / i);
                }
            }
        }
        return divisors;
    }

    // 打乱函数
    long shift(long x) {
        x ^= x << 13;
        x ^= x >>> 7;
        x ^= x << 17;
        return x;
    }

    long getComp(long a, long b) {
        long up = 1;
        long down = 1;
        for (int i = 1; i <= b; i++) {
            up = (up * ((a+b-i) % mod)) % mod;
            down = (down * i) % mod;
        }
        return (up * power(down, mod-2, mod)) % mod;
    }

    long power(long x, long power, long mod) {
        if (power <= 0) return 1;
        if (power <= 1) return x;
        long res = power(x, power/2, mod);
        res = (res * res) % mod;
        if (power % 2 == 1) res = (res * x) % mod;
        return res % mod;
    }
    
}

class Kattio extends PrintWriter {
    private final InputStream in;
    private final boolean closeInput;

    private final byte[] buffer = new byte[1 << 16];
    private int ptr = 0, len = 0;

    // -2: 没有缓存；-1: EOF；其他: 缓存的一个字节
    private int peek = -2;

    public Kattio() {
        this(System.in, System.out);
    }

    public Kattio(InputStream input, OutputStream output) {
        super(new BufferedWriter(new OutputStreamWriter(output)));
        this.in = input;
        this.closeInput = false; // 默认不主动关 System.in
    }

    public Kattio(String inputFile, String outputFile) throws IOException {
        super(new BufferedWriter(new FileWriter(outputFile)));
        this.in = new FileInputStream(inputFile);
        this.closeInput = true;
    }

    private int readByte() {
        if (ptr >= len) {
            try {
                len = in.read(buffer);
                ptr = 0;
            } catch (IOException e) {
                throw new UncheckedIOException(e);
            }
            if (len <= 0) return -1;
        }
        return buffer[ptr++] & 0xff; // 修正 signed byte 问题
    }

    private int nextByte() {
        if (peek != -2) {
            int c = peek;
            peek = -2;
            return c;
        }
        return readByte();
    }

    private int skipBlank() {
        int c;
        do {
            c = nextByte();
        } while (c != -1 && c <= ' ');
        return c;
    }

    public boolean hasNext() {
        int c = skipBlank();
        if (c == -1) return false;
        peek = c;
        return true;
    }

    public String next() {
        int c = skipBlank();
        if (c == -1) throw new NoSuchElementException("No more tokens");

        StringBuilder sb = new StringBuilder();
        while (c > ' ') {
            sb.append((char) c);
            c = nextByte();
        }
        return sb.toString();
    }

    public int nextInt() {
        long x = nextLong();
        if (x < Integer.MIN_VALUE || x > Integer.MAX_VALUE) {
            throw new NumberFormatException("int overflow: " + x);
        }
        return (int) x;
    }

    public long nextLong() {
        int c = skipBlank();
        if (c == -1) throw new NoSuchElementException("No more tokens");

        int sign = 1;
        if (c == '-' || c == '+') {
            sign = (c == '-') ? -1 : 1;
            c = nextByte();
        }

        if (c < '0' || c > '9') {
            throw new NumberFormatException("Invalid long");
        }

        long val = 0;
        while (c > ' ') {
            if (c < '0' || c > '9') {
                throw new NumberFormatException("Invalid long");
            }
            val = val * 10 + (c - '0');
            c = nextByte();
        }

        return sign == 1 ? val : -val;
    }

    public double nextDouble() {
        return Double.parseDouble(next());
    }

    public int[] nextIntArray(int n) {
        int[] a = new int[n];
        for (int i = 0; i < n; i++) a[i] = nextInt();
        return a;
    }

    public long[] nextLongArray(int n) {
        long[] a = new long[n];
        for (int i = 0; i < n; i++) a[i] = nextLong();
        return a;
    }

    @Override
    public void close() {
        flush();
        super.close();
        if (closeInput) {
            try {
                in.close();
            } catch (IOException e) {
                throw new UncheckedIOException(e);
            }
        }
    }
```
